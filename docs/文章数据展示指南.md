# 内容数据展示指南

> 说明本项目如何从后端 API 获取并展示游戏、盒子、攻略等内容数据

## 内容路由结构

| 路由 | 说明 | API |
|------|------|-----|
| `/games` | 游戏库列表 | `GET /api/public/games` |
| `/games/[id]` | 游戏详情 | `GET /api/public/games/{id}` |
| `/games/category/[id]` | 游戏分类 | `GET /api/public/games/categories` |
| `/boxes` | 盒子大全列表 | `GET /api/public/boxes` |
| `/boxes/[id]` | 盒子详情 | `GET /api/public/boxes/{id}` |
| `/content/guides/[slug]` | 攻略详情 | `GET /api/public/articles/{slug}` |
| `/content/reviews/[slug]` | 评测详情 | `GET /api/public/articles/{slug}` |
| `/news/[slug]` | 资讯详情 | `GET /api/public/articles/{slug}` |
| `/search` | 搜索 | `GET /api/public/search` |

所有路由均有多语言版本（`/zh-TW/games`、`/en-US/games` 等）。

## 快速启动

### 步骤1：确保后端服务运行

```bash
# 在项目根目录（game-box）执行
.\mvnw.cmd spring-boot:run -pl ruoyi-admin
```

或使用 VS Code 任务：`Ctrl+Shift+P` → `Tasks: Run Task` → `Spring Boot: 启动后端`

### 步骤2：配置环境变量

创建 `Next-web/.env.local`：

```env
NEXT_PUBLIC_API_URL=http://localhost:8080
NEXT_PUBLIC_SITE_ID=1
```

### 步骤3：启动前端

```bash
cd Next-web
pnpm dev
```

访问 http://localhost:3000 即可看到游戏和内容数据。

## API 客户端使用

所有 API 调用统一使用 `src/lib/api.ts` 的 `ApiClient`，无需直接操作 fetch：

```typescript
import ApiClient from '@/lib/api'

// 获取游戏列表
const response = await ApiClient.getGames({
  locale: 'zh-CN',
  pageNum: 1,
  pageSize: 20,
})
const games = response.rows || []

// 获取游戏详情
const game = await ApiClient.getGameDetail(id, 'zh-CN')

// 获取盒子列表
const boxResponse = await ApiClient.getBoxes({ locale: 'zh-CN', pageNum: 1, pageSize: 20 })

// 获取攻略/资讯文章
const article = await ApiClient.getArticleDetail(slug, 'zh-CN')

// 搜索
const searchResult = await ApiClient.searchArticles({ q: 'keyword', locale: 'zh-CN' })
```

## 数据流程

```
用户访问 /games
  ↓
Next.js 服务端渲染（SSR/ISR）
  ↓
调用 ApiClient.getGames({ locale: 'zh-CN' })
  ↓
GET http://localhost:8080/api/public/games?pageNum=1&pageSize=20&siteId=1
  ↓
后端返回 JSON 数据
  ↓
页面渲染游戏卡片列表
```

## 多语言数据获取

每个页面路径中包含 `locale` 参数，会自动传给 API：

```typescript
// app/[locale]/games/page.tsx
export default async function GamesPage({ params }: { params: { locale: string } }) {
  const response = await ApiClient.getGames({ locale: params.locale as Locale })
  // ...
}
```

## 常见问题

### 页面无数据显示

1. 确认后端服务运行中：访问 `http://localhost:8080/api/public/games`
2. 检查 `.env.local` 中 `NEXT_PUBLIC_API_URL` 是否正确
3. 查看浏览器控制台（F12）的网络请求和错误

### 图片无法显示

后端返回的图片 URL 可能是相对路径或使用了 CDN 域名，需确认 `next.config.js` 的 `images.domains` 配置包含图片来源域名，或当前已设置 `unoptimized: true`（默认）。

### ISR 缓存刷新

生产环境页面有 ISR 缓存，需要刷新数据时调用：

```bash
POST /api/revalidate?path=/games&secret=your-secret
```
