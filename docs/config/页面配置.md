# 页面配置目录说明

## 📁 目录结构

```
src/config/pages/
├── index.ts            # 统一导出入口
├── home.ts             # 首页配置 (/)
├── games.ts            # 游戏库配置 (/games, /games/[id])
├── boxes.ts            # 盒子大全配置 (/boxes, /boxes/[id])
├── content.ts          # 内容中心配置 (/content/...)
├── news.ts             # 资讯配置 (/news, /news/[slug])
└── search.ts           # 搜索页配置 (/search)
```

## 🎯 设计理念

### 按路由组织配置
每个页面（或页面组）对应一个配置文件，便于查找和维护。

### 数据与 UI 分离
- **配置层** (`config/pages/`): 管理页面的文案、标题、描述等 UI 配置
- **数据层** (`lib/`): 数据获取逻辑（Server Actions / Fetch API）
- **展示层** (`app/`): 纯展示组件，消费配置和数据

### 类型安全
所有配置都有完整的 TypeScript 类型定义，确保配置正确性。

## 📋 配置文件详解

### 1. home.ts — 首页

**路由:** `/`

```typescript
{
  hero: {
    badge: string             // 顶部徽章文本
    title: string             // 主标题
    highlightText: string     // 高亮词
    description: string[]     // 描述文本（多行）
    primaryButton: { text, href }
    secondaryButton: { text, href }
  },
  stats: [{ label, value, icon }],
  sections: { ... },          // 内容区块
  metadata: { title, description, keywords }
}
```

**使用:**
```typescript
import { getHomePageConfig } from '@/config'
const config = getHomePageConfig('zh-CN')
```

### 2. games.ts — 游戏库

**路由:** `/games`, `/games/[id]`

**配置项:** Hero 区域、筛选器（游戏类型/平台）、排序选项、卡片展示、SEO 元数据

### 3. boxes.ts — 盒子大全

**路由:** `/boxes`, `/boxes/[id]`

**配置项:** Hero 区域、筛选器（平台/折扣力度）、排序选项、卡片展示、SEO 元数据

### 4. content.ts — 内容中心

**路由:** `/content/guides`, `/content/reviews`, `/content/topics` 及各详情页

**配置项:** 各内容类型（攻略/评测/专题）独立标题和描述、分类筛选器、详情页目录/侧边栏、SEO 元数据

### 5. news.ts — 资讯

**路由:** `/news`, `/news/[slug]`

**配置项:** Hero 区域、分类筛选、列表展示配置、SEO 元数据

### 6. search.ts — 搜索页

**路由:** `/search`

**配置项:** 搜索框占位符、热门关键词、最近搜索设置、搜索结果展示、SEO 元数据

## 🚀 使用方式

### 在页面组件中消费配置

```typescript
// app/games/page.tsx
import { getGamesPageConfig } from '@/config'

export default async function GamesPage() {
  const config = getGamesPageConfig('zh-CN')
  return <h1>{config.hero.title}</h1>
}
```

### 修改配置文本

直接编辑对应配置文件：

```typescript
// src/config/pages/games.ts
export function getGamesPageConfig(locale: string) {
  return {
    hero: {
      title: '游戏库',      // 👈 修改标题
      description: '...',  // 👈 修改描述
    },
    // ...
  }
}
```

### 多语言支持

配置函数接受 `locale` 参数，返回对应语言版本：

```typescript
const zhConfig = getGamesPageConfig('zh-CN')   // 简体中文（默认）
const twConfig = getGamesPageConfig('zh-TW')   // 繁体中文
const enConfig = getGamesPageConfig('en-US')   // 英文
```
