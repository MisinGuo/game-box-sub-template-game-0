### Step 1 — 整理子站游戏/盒子列表（10 分钟）

**目标**：建立「可推荐游戏」+「主站盒子链接」对照表，后续所有文章只能推荐此列表内的游戏。

**操作路径**：
- 后台 → 游戏管理 → 游戏列表 → 查看模式「**关联网站**」→ 网站 **SITE_ID=8** → 筛选卡牌/策略 → 记录游戏名 + ID
- 后台 → 游戏管理 → 盒子列表 → 查看模式「**关联网站**」→ 网站 **SITE_ID=8** → 记录盒子名 + ID（主站链接：`/boxes/[id]`）

> ⚠️ AI 文章中推荐的游戏必须全部来自此列表，不得推荐子站未录入的游戏。

### Step 3 — 关键词挖掘（10 分钟）

**调用**：[M2-游戏关键词表.md](./seo/materials/M2-游戏关键词表.md) — 关键词挖掘提示词

- **输出**：15~30 个真实搜索词（直接需求词 / 痛点词 / 竞品词，含搜索量级）

```
你是一名 SEO 关键词研究员，熟悉百度、TapTap 的手游玩家搜索行为。

请针对以下游戏，生成用户在搜索引擎中真实可能输入的关键词：

目标游戏：率土之滨

请按三类各给出 8~10 个关键词：

【直接需求词】例如：率土之滨好玩吗、类似率土之滨的游戏、率土之滨平替推荐
【痛点词】例如：率土之滨太肝了、比率土之滨肝度低的游戏、上班族能玩的三国手游
【竞品词】例如：类似率土之滨的卡牌手游、率土之滨同类游戏推荐、率土之滨换游玩什么

要求：
1. 每个词必须是用户真实可能输入的自然语言（5~15字）
2. 标注预估搜索量级：高/中/低/待查
3. 输出格式（表格）：类型 | 关键词 | 预估搜索量级
```

```
   - 内链占位：【游戏详情链接占位/games/ID】【主站盒子链接占位/boxes/ID】

   - [核心关键词] 在 H1/H2 标题及正文首段自然出现 3~5 次

   - 截流目标游戏引言结尾处：
```

### Step 7.5 — Meta Description 人工微调（每篇 3 分钟）

**调用**：[M9-Meta描述.md](./seo/materials/M9-Meta描述.md)

> Meta Description 直接决定搜索结果点击率（CTR），务必手动撰写，禁止用 AI 原文截取。

**撰写规则**：
- 总长度：110~150 字符（中文约 50~70 字）
- 前 50 字必须包含**「省钱/护肝」核心钩子**
- 必须包含目标关键词
- 以行动号召（CTA）结尾

**模板参考**（发布时按实际内容替换）：

```
率土之滨玩累了？2026年老玩家都在转战这几款[核心关键词]！主打[肝度低/0.1折]，
策略性不减，更适合上班族。点击查看最新[核心关键词]平替清单。
```

## 后续扩量判断（本批发布 1 周后观察）

| 指标 | 判断标准 |
|---|---|
| 抓取 | Google Search Console 显示文章已被抓取 |
| 展现 | 有曝光量（哪怕很少） |
| 点击 | 有点击，说明标题和关键词匹配到了搜索意图 |

满足「已抓取 + 有展现」即可扩量至每天 5~8 篇，并增加第 2 款截流目标游戏（建议：文明与征服）。

**GSC 数据决策树**：

| 观察结果 | 行动 |
|---|---|
| 有展现，无点击 | 优化标题：加更多数字（"0.1折"、"2026最新"），或将年份提到标题最前 |
| 无展现（未被索引） | 检查 sitemap 是否已提交 GSC；在子站首页为该文章增加曝光入口 |
| 有点击，无转化 | 核查推荐游戏折扣是否真实有效；检查 `/games/[id]` 落地页加载速度 |
| 排名 11~30 名 | 在文章中补充 200~400 字新内容，增加一个新 H2 段落（如：详细避坑指南） |
