## 各页面详细说明

### 首页 `/`

**文件**：`src/app/page.tsx` / `src/app/[locale]/page.tsx`  
**功能**：站点入口，展示热门游戏、精选盒子、近期资讯  
**关键内容**：Hero 区域突出"折扣游戏"卖点，导入 `/01zhe/`、`/boxes/`、`/rank/` 等核心转化路径

---

### 游戏库 `/games`

**文件**：`src/app/[locale]/games/page.tsx`  
**功能**：全量游戏列表，支持分类筛选、关键词搜索  
**内链策略**：每个游戏卡片链接至 `/games/[id]`，页面顶部推荐 `/boxes/` 入口

#### 游戏详情 `/games/[id]`

**文件**：`src/app/[locale]/games/[id]/page.tsx`  
**功能**：单款游戏的完整介绍，包含：介绍、截图、礼包、盒子折扣对比、相关攻略  
**SEO重点**：title 格式为 `游戏名 - 盒子折扣 | 5AWYX`，JSON-LD `VideoGame` schema  
**转化路径**：点击盒子折扣 → `/boxes/[id]` 详情页

#### 游戏分类 `/games/category/[id]`

**文件**：`src/app/[locale]/games/category/[id]/page.tsx`  
**功能**：按品类展示游戏列表（如"卡牌"、"RPG"、"策略"），辅助品类词 SEO  
**内链**：分类页链接回 `/games/[id]`，同时推荐同类盒子

---

### 盒子大全 `/boxes`

**文件**：`src/app/[locale]/boxes/page.tsx`  
**功能**：所有游戏盒子列表，支持按折扣力度/特色筛选  
**SEO重点**：核心转化页，必须在首页、游戏详情页、资讯文章中大量内链此处

#### 盒子详情 `/boxes/[id]`

**文件**：`src/app/[locale]/boxes/[id]/page.tsx`  
**功能**：单个盒子的详细介绍：折扣率、支持游戏、安全资质、注册步骤  
**转化路径**：此页包含推广链接（CPS/CPA），是最终转化終点

---

### 0.1折专区 `/01zhe`

**文件**：`src/app/[locale]/01zhe/page.tsx`  
**功能**：汇聚标注"0.1折"属性游戏 + 相关攻略文章  
**SEO重点**：核心流量关键词"0.1折手游"落地页，需大量从资讯文章内链至此  

> 此页是站内核心 SEO 落地页，所有问答型、截流型资讯文章都应内链到 `/01zhe/`。

---

### 排行榜 `/rank`

**文件**：`src/app/[locale]/rank/page.tsx`  
**功能**：按下载量排名的手游排行榜（TOP 100）  
**SEO重点**："手游排行榜"类关键词专用落地页  
**内链**：每个榜单游戏链接至 `/games/[id]`，页面推荐盒子折扣入口

---

### 资讯详情 `/news/[id]`

**文件**：`src/app/[locale]/news/[slug]/page.tsx`  
**对应后台**：SITE_ID=`1`，Section Slug=`news-section`  
**功能**：单篇资讯文章展示，包括问答型、截流型、安全顾虑型、话题型、新游预告型  
**SEO字段来源**：

- `<meta description>` ← 后台「摘要/描述」字段

**内链要求**：必须包含 `/games/[id]`、`/01zhe/` 或 `/boxes/[id]`

### 搜索 `/search`

**文件**：`src/app/[locale]/search/page.tsx`  
**功能**：全站搜索，包含游戏和文章结果  
**渲染**：纯 CSR（客户端搜索），不被收录为独立 SEO URL  
**注意**：生产环境若搜索结果为空，检查后端 CORS 和 API 地址配置

---

### 标签聚合页 `/tag/[tag]`

**文件**：`src/app/[locale]/tag/[tag]/page.tsx`  
**渲染方式**：动态渲染（`force-dynamic`，tag 太多不预生成静态页）  
**功能**：按标签词聚合展示匹配的游戏（最多40个） + 文章（最多18篇）  

**页面生成逻辑**：
- 从 URL 取出 `[tag]` 值（如 `三国卡牌`）
- 以此 tag 为关键词分别调用 `searchGames` 和 `searchArticles` API
- 展示匹配结果，提供跳转到游戏详情页和文章详情页的链接

**SEO 元数据**：
```
title: 「{tag}」相关游戏与攻略 - 我爱玩游戏网
description: 查看所有标签为「{tag}」的手游与攻略文章，并对比游戏盒子折扣找到最优惠的玩法。
keywords: {tag},{tag}手游,{tag}攻略,{tag}游戏盒子
```

**数据来源**：文章的「关键词」字段（逗号分隔）→ 解析为 `tags: string[]` 数组，用于：
1. `<meta name="keywords">` — 文章详情页 SEO 关键词
2. Open Graph `article:tag` — 社交分享标签
3. JSON-LD `Article.keywords` — 结构化数据
4. `/tag/[tag]` 聚合页 — 长尾 SEO 落地页

**填写建议**：至少填写「游戏名 + 品类词 + 意图词」，例如：  
`三国志战略版,三国卡牌,三国游戏攻略,游戏盒子折扣`

> tag 聚合页的主要价值在于覆盖长尾 SEO 关键词（如"三国卡牌"、"0.1折仙侠"），而非主要转化入口。URL 形如 `/tag/三国卡牌` 会被搜索引擎收录。
