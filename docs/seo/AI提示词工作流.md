# AI 内容生产提示词工作流

> 本文档定义从"找关键词"到"出文章"的完整 5 步 AI 提示词流程。
> 每一步的输出是下一步的输入，可以串行执行，也可以人工卡每步质量后再继续。
> **核心原则：先对准搜索需求，再包装内容转化。没有搜索量的标题，内容再好也没人找到。**

---

## 流程总览

```
Step -1：截流目标游戏筛选  →  得到：10 款候选衰退期大游（含品类/衰退原因/截流角度）
    ↓
Step 0：关键词挖掘        →  得到：该游戏 15~30 个真实搜索词（带流量判断）
    ↓
Step 1：痛点挖掘          →  得到：该游戏的 12~15 条专属痛点清单
    ↓
Step 2：角度矩阵生成      →  得到：N 篇文章任务卡（关键词 × 角度 × 标题）
    ↓
Step 3：去重筛选          →  得到：今日待写的 2~5 个任务（流量×转化双优先）
    ↓
Step 4：文章正文生成      →  得到：关键词密度达标、可直接发布的完整文章
```

---

## Step -1：截流目标游戏筛选提示词

**什么时候用：** 启动新一轮内容生产，还没有选定截流目标游戏时。输出候选列表后，人工对照子站已录入游戏品类确认即可。

**输入：** 子站品类定位（例如：卡牌/策略）

---

```
你是一名手游市场分析师，熟悉国内安卓手游的生命周期规律和玩家社区动态。

请列出目前处于「衰退期」的卡牌/策略类手游，筛选标准如下：
1. 曾是国内热门游戏（百度指数或 TapTap 评价数峰值较高）
2. 当前仍有稳定搜索量，但运营商已大幅缩减广告投入
3. 玩家社区内频繁出现「玩腻了/找平替/氪不动了」等讨论

请列出 10 款符合条件的游戏，每款输出：
游戏名 | 品类（细分） | 衰退原因（一句话） | 截流角度建议（一句话）

要求：
- 只列卡牌/策略品类
- 优先列国产手游
- 截流角度建议要具体（例如：主打「上班族肝度低的平替」方向）
```

**人工确认（1 分钟）**：拿到候选列表后，对照子站已录入游戏列表，优先选品类可承接（子站有对应推荐游戏）的目标游戏。

## Step 0：关键词挖掘提示词（首要步骤）

**什么时候用：** 遇到一款新的目标游戏，**先于所有步骤运行**。只有确认搜索需求存在，后续内容生产才有意义。

**输入：** 游戏名

---

### 三类关键词（必须覆盖）

| 类型 | 说明 | 示例 |
|---|---|---|
| **直接需求词** | 用户搜该游戏相关内容的变体，流量最大 | `率土之滨平替`、`类似率土之滨的游戏`、`率土之滨好玩吗` |
| **痛点词** | 用户带负面意图的搜索，转化率最高 | `三国卡牌太肝`、`不氪金能玩的卡牌手游`、`上班族卡牌手游推荐` |
| **竞品词** | 以热门游戏名为锚的长尾词，SEO 核心流量入口 | `类似王者荣耀的游戏`、`原神同类卡牌手游`、`文明与征服平替` |

---

```
你是一名 SEO 关键词研究员，熟悉百度、TapTap 的手游玩家搜索行为。

请针对以下游戏，生成用户在搜索引擎中真实可能输入的关键词：

目标游戏：【填入游戏名（截流目标，即热门大游）】

请按三类各给出 8~10 个关键词：

【直接需求词】用户搜索该游戏相关内容的变体，例如：
- [游戏名]好玩吗
- [游戏名]值得玩吗
- [游戏名]2026还能玩吗
- 类似[游戏名]的游戏
- [游戏名]平替推荐

【痛点词】用户对该游戏产生厌倦/不满时会搜的词，例如：
- [游戏名]太肝了
- 比[游戏名]肝度低的游戏
- 三国卡牌不氪金能玩多久
- 上班族能玩的三国手游

【竞品词】以该游戏名为对比锚点的长尾词，例如：
- 类似[游戏名]的卡牌手游
- [游戏名]同类游戏推荐
- [游戏名]换游玩什么

要求：
1. 每个词必须是用户真实可能输入的自然语言（5~15字）
2. 标注预估搜索量级：高（月均万次+）/ 中（月均千次）/ 低（月均百次）/ 待查
3. 输出格式（表格）：类型 | 关键词 | 预估搜索量级
```

---

### 人工验证方法（建议对"中/高"量级词快速验一遍）

| 工具 | 操作 | 时间 |
|---|---|---|
| **百度搜索框下拉** | 输入词，出现联想即有搜索量，直接采用 | 1 分钟 |
| **百度指数** | `index.baidu.com` 查趋势是否仍活跃 | 2 分钟 |
| **5118 / 爱站** | 批量导出相关长尾词，按搜索量排序，一次填满素材卡 | 5 分钟 |
| **竞品文章标题** | 百度搜游戏名，看排名前 5 文章用了哪些词 | 2 分钟 |

**输出：** 每款游戏 15~30 条关键词表，存入素材卡「搜索关键词」栏，Step 2 生成标题时直接调用。

## Step 1：游戏痛点挖掘提示词

**什么时候用：** 遇到一款新的目标游戏，首次使用前运行一次，结果存入素材库。

**输入：** 游戏名（可一次放 1~3 款）

---

```
你是一名资深手游玩家和游戏社区运营，熟悉国产手游的常见问题和玩家流失规律。

请分析以下游戏，列出玩家最常见的 12~15 个离开理由或强烈痛点：

游戏名：【填入游戏名】

要求：
1. 每条痛点必须具体，不能泛泛而谈
   - 错误示例："游戏太肝"
   - 正确示例："每天必须完成12个日常任务，公会战强制在线否则被踢，活动期间需要连续签到7天一天不缺"
2. 每条痛点后注明：这个痛点导致哪类玩家流失（例："导致上班族、轻度玩家大量流失"）
3. 按以下维度分组输出：
   - 【时间压力】需要每天固定在线、肝日常等
   - 【付费压力】氪金门槛、月卡/礼包性价比差等
   - 【社交绑架】公会压力、强制组队等
   - 【运营问题】版本更新慢、BUG多、客服差、停服风险等
   - 【竞技失衡】充值玩家碾压、零氪上限低等
   - 【画质/体验】UI老旧、操作卡顿、服务器差等

输出格式：
## 【维度名称】
1. [具体痛点描述] → 导致[流失用户画像]
2. ...
```

---

**输出示例（率土之滨）：**

```
## 【时间压力】
1. 州郡战期间需要每天至少2小时在线盯时间差，下班族根本跟不上节奏 → 导致有工作的成年玩家大量流失
2. 公会战需要固定时间集结，迟到会被踢出公会影响整体资源获取 → 导致无法保证固定时间的玩家流失

## 【社交绑架】
3. 一旦加入强公会，被迫参与各类会战，退会则失去核心资源获取渠道 → 导致独狼玩家和社恐玩家流失
...
```

---

> **存档方式：** 每款游戏的痛点列表存成一个文本文件或表格行，后续 Step 2/Step 4 直接调用。

### ⚠️ 问题1：AI对冷门/小众游戏了解有限，痛点会不准确

**现象：** AI对中小型游戏的训练数据少，会用"通用手游痛点"凑数，导致每款游戏都差不多。

**判断方法：** 跑完Step 1后，如果痛点描述里没有出现该游戏**独有的系统或机制名称**（如特定副本名、特定活动名），说明AI在瞎编，不可用。

**解决方案：使用评论辅助版**

遇到AI不熟悉的游戏，改用以下提示词，把TapTap/App Store的真实玩家评论喂给AI：

```
我有以下来自玩家的真实评论，请从中提取玩家离开这款游戏的真实原因：

游戏名：【填入游戏名】

玩家评论（从TapTap或App Store差评区复制10~20条）：
【粘贴原始评论，无需整理格式】

请：
1. 将重复出现的抱怨归类整理，合并同类项
2. 每类痛点用一句具体描述（必须包含评论中出现过的具体细节）
3. 标注这条痛点在评论中出现的频率：高/中/低
4. 维度分类：时间压力 / 付费压力 / 社交绑架 / 运营问题 / 竞技失衡 / 画质体验

输出格式与 Step 1 标准版相同。
```

> **获取评论的方法：** 打开 TapTap → 搜索游戏 → 点"评价" → 按"最新"或"差评"排序 → 复制粘贴 10~20 条

## Step 2：角度矩阵生成提示词

**什么时候用：** 拿到 Step 1 的痛点清单后，生成这款游戏可以写的所有文章任务。

**输入：** Step 0 的搜索关键词清单 + Step 1 的痛点清单 + 游戏名

---

```
你是一名 SEO 内容策划，专门为手游推广站生产高转化文章选题。

我有一款游戏的完整数据如下：

游戏名：【填入游戏名】
真实搜索关键词（来自 Step 0，已验证有搜索量）：
【粘贴 Step 0 的输出关键词表】

痛点清单（来自 Step 1）：
【粘贴 Step 1 的输出】

⚠️ 重要规则：每篇文章的标题核心词必须来自上面"真实搜索关键词"中的某个词，不得自行发明搜索词。

请根据以上内容，生成这款游戏可以写的所有文章角度，按以下 8 类分别列出：

1. 【截流型】标题含"平替/同类/喜欢XX的玩家"，目标拦截搜索该游戏的用户
2. 【痛点解决型】标题含"太肝/太氪/不想花钱/上班族能玩吗"，精准命中流失意图
3. 【充值避坑型】标题含"月卡/首充/充值值不值/省钱方案"，拦截消费决策
4. 【平替推荐型】标题含"有没有类似/更好玩的同类游戏"，收割换游意图
5. 【入坑决策型】标题含"值得玩吗/好玩吗/2026年还能玩吗"，帮助观望用户决策
6. 【零氪攻略型】标题含"零氪玩到哪步/不充钱能玩多久"，服务低付费用户
7. 【时间友好型】标题含"上班族/每天几分钟/不用花太多时间"，收割时间受限用户
8. 【跑路/停服型】标题含"还值得玩吗/会停服吗/稳不稳"，拦截犹豫迁移的玩家

每类输出 2~3 个具体标题，要求：
- 标题中必须代入真实的游戏名
- 标题核心词必须来自 Step 0 的搜索词（直接包含或自然变体）
- 同时说明：① 核心论点（不超过20字）② 对应的 Step 0 关键词

输出格式（表格）：
| 角度类型 | 文章标题 | 核心论点 | 对应搜索词 |
|---|---|---|---|
```

---

**输出结果即为该游戏的"文章任务卡"，每行 = 一篇文章。**

## Step 3：每日任务筛选提示词

**什么时候用：** 每天开始生产前，用这个提示词从素材库里快速挑出今天要写的 3~5 篇。

**输入：** 多款游戏的角度矩阵 + 已发布去重索引（见格式说明）

### ⚠️ 问题2：已发布清单随时间膨胀

**原方案** 把已发布文章完整标题都粘进来，发布100篇后每次提示词会有几千字历史记录。

**修复：改用"游戏名|角度类型"二元组索引**，每条只需 10~15 个字，100篇只占1000字：

```
已发布去重索引（格式：游戏名|角度类型，每行一条）：
率土之滨|截流型
率土之滨|痛点解决型
文明与征服|充值避坑型
三国志战略版|截流型
...
```

完整提示词如下：

---

```
我有以下手游文章待写任务库：

【粘贴多款游戏的角度矩阵表格，或列出游戏名×角度组合】

已发布去重索引（格式：游戏名|角度类型，每行一条，无则填"暂无"）：
【粘贴二元组索引】

请帮我从待写任务中，**排除已发布的游戏×角度组合**，然后按以下优先级选出今天最值得写的 5 篇：

优先级规则（流量 × 转化 双维度）：
1. 有明确停服/回归/联动事件触发 → 最高优先（时效窗口短，立刻写）
2. Step 0 关键词有搜索量 且 为截流型/平替推荐型 → 高优先（流量×转化双赢）
3. Step 0 关键词有搜索量 且 为充值避坑型 → 高优先（商业转化意图极强）
4. 痛点解决型（对应关键词在 Step 0 中能找到）→ 中优先
5. 入坑决策型 + 零氪攻略型 → 正常优先
6. 无对应搜索词的时间友好型 / 跑路停服型 → 备选

判断"有搜索量"：对应角度在 Step 0 关键词表中能找到直接词或变体词。

今天的游戏偏向（如有）：【可选填，例如"今天主要写三国题材"或"不限"】

输出格式（5行表格）：
| 优先级 | 游戏名 | 角度类型 | 目标关键词 | 推荐标题 | 写作重点提示（一句话） |
|---|---|---|---|---|---|
```

## Step 4：文章正文生成提示词

**什么时候用：** 拿到 Step 3 的单个任务后，生成完整文章。这是最终出稿步骤。

**输入：** 游戏名 + 角度类型 + 标题 + 该游戏的专属痛点（从 Step 1 取对应内容）

---

```
你是一名手游内容创作者，专为手游攻略站写推荐文章，文风口语化、有说服力、不夸张。

任务信息：
- 游戏名：【填入目标截流游戏名，即玩家正在搜的热门游戏】
- 文章角度：【填入角度类型，如"截流型/痛点解决型/充值避坑型"】
- 文章标题：【填入 Step 2/3 生成的标题】
- 该游戏玩家的核心痛点（必须写进文章体现差异化）：
  【粘贴 Step 1 中该游戏对应的 3~5 条具体痛点】

推荐的替代游戏（**只能从以下子站收录的游戏中选择，不得推荐其他游戏**）：
【从"子站游戏卡片库"中选2~3款，复制对应卡片粘贴到这里】
格式示例：
- 三国群英传-霸王之业：回合制策略卡牌，主打低氪金门槛，每日任务15分钟即可完成
- 少年三国志：零氪：武将碎片可纯养成获取，竞技场匹配段位制不强制PVP

内链占位符（请在对应位置插入，不要删除，发布前人工替换为真实URL）：
- 游戏A详情页：{{GAME_A_URL}}
- 游戏B详情页：{{GAME_B_URL}}
- 游戏A盒子页：{{BOX_A_URL}}
- 游戏B盒子页：{{BOX_B_URL}}

折扣信息规则（⚠️ 重要）：
- 不得自行编造任何折扣数字
- 折扣数据一律写占位符：[折扣:X折待填] 或 [首充金额:XX元待填]
- 文章完成后人工核实后替换

文章结构要求：
1. 引言（50字内）：认可目标游戏的优点，但点出玩家普遍遇到的[上面列的具体痛点]
2. 推荐游戏列表（主体，每款约150字）：
   - 游戏名 + 一句话定位
   - 针对上述痛点的解决方案（具体说这款游戏怎么解决了这个痛点）
   - 盒子首充折扣信息
   - 插入内链占位符
3. 对比总结表（可选）：3列 = 游戏名/核心优势/折扣率
4. 结尾引导（30字）：引导去主站找盒子折扣或子站看更多攻略

字数：800~1200字
风格：口语化，像玩家写给玩家，不要营销腔
关键词密度要求（⚠️ SEO 关键，不达标百度不收录）：
- 目标关键词（来自 Step 0，如"率土之滨平替"）：标题中出现 1 次，正文中自然出现 3~5 次
- 长尾变体词（如"类似率土之滨的游戏""率土之滨同类手游"等变体）：正文中出现 2~3 次
- H2/H3 小标题中至少包含 1 个关键词变体
- 不得堆砌，每次出现必须在语义通顺的句子中

输出：完整文章正文（Markdown格式）
```

## 使用说明

### 每周素材积累（15分钟）

```
1. 选 3~5 款本周目标游戏
2. 每款跑一次 Step 0，挖掘真实搜索词（存入素材卡「搜索关键词」栏）
3. 每款跑一次 Step 1，提取痛点
4. 每款跑一次 Step 2，基于 Step 0 关键词生成角度矩阵
5. 把结果存入游戏素材库（见《内容追踪矩阵》）
```

### 每日生产（按需）

```
1. 跑一次 Step 3，从库里选出今日 5 个任务
2. 每个任务跑一次 Step 4，生成文章
3. 人工审核：确认推荐游戏存在于子站，数据无明显错误
4. 后台发布：SITE_ID=8，Section=news-section
5. 在追踪矩阵中标记为 ✅
```

### 质量红线（Step 4 审核必查）

| 检查项 | 说明 |
|---|---|
| 痛点是否具体 | 文章中的游戏问题描述必须具体（不能只是"太肝"） |
| 推荐游戏是否子站有 | 不能推荐子站没有收录的游戏 |
| 内链是否填充 | 替换所有 `{{URL}}` 和 `[折扣:X折待填]` 占位符为真实数据 |
| 折扣数据是否真实 | 数字必须来自主站后台实际价格，不接受AI编造 |
| 核心论点是否差异化 | 每篇文章的核心推荐理由必须不同，不能是换名字的模板 |

## 附：角度类型速查

| 角度类型 | 目标用户 | 转化点 | 内链重点 |
|---|---|---|---|
| 截流型 | 正在玩/搜大热门游戏的用户 | 同类游戏推荐 → 子站 | 游戏详情页 + 盒子页 |
| 痛点解决型 | 已经不满当前游戏的用户 | 解决痛点的替代游戏 | 游戏详情页 + 盒子页 |
| 充值避坑型 | 准备充钱的用户 | 官服 → 盒子首充转化 | 主站盒子页（核心） |
| 平替推荐型 | 主动在找换游方向的用户 | 推荐列表 → 子站游戏 | 多个游戏详情页 |
| 入坑决策型 | 观望中、未决策的用户 | 打消顾虑 → 降低门槛 | 盒子页（零门槛试玩） |
| 零氪攻略型 | 不想花钱但想玩的用户 | 推动下载 + 盒子注册礼包 | 盒子注册礼包链接 |
| 时间友好型 | 轻度玩家、上班族 | 推荐低时间成本游戏 | 游戏详情页 |
| 跑路/停服型 | 担心游戏稳定性的用户 | 推荐稳定运营游戏 | 游戏详情页 + 主站盒子 |
