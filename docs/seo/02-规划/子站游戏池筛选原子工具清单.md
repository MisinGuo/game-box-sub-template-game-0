# 子站游戏池筛选原子工具清单

> 对应方法论：《子站游戏池筛选方法论》  
> 目标：把“榜单扫描 → 三轨清洗 → 词根碰撞 → 7仓位落位 → 扩展游戏池”的完整流程拆成可配置、可复用的原子工具。

---

## 一、总流程

```text
阶段1：三源合流全景扫描
  ↓
阶段2：候选池合并去重 + 来源打标 + 三轨分流
  ├─ 新游绿道：免 AI 粗筛，直接进入搜索词验证
  ├─ 高转平替：AI 轻筛 + 痛点词确认 + 盒子库存匹配
  └─ 大盘老游：AI 粗筛 + 搜索需求验证 + 内容可行性判断
  ↓
阶段3：三轨汇合·词根碰撞
  ↓
阶段4：7仓位精准落位 + 承接路径规划
  ↓
阶段5：扩展游戏池 + 最终检查清单
```

## 三、阶段1：三源合流全景扫描

> **目标：** 从 4 个免费数据源采集榜单，形成候选池。
> **数据源优先级：** Tap（最全） > 好游快爆（下载量+时间线） > B站（二次元补充） > 微信小游戏（下沉市场）

---

### 平台 A：TapTap（`taptap.cn`）

**域名：** `https://www.taptap.cn`
**渲染方式：** SSR 直出 HTML，cheerio 可直接解析
**翻页：** `?page={page}` 参数，每页约 20 条
**更新频率：** 每 20 分钟

#### `seo_game_rank_taptap_fetcher`

**用途：** 抓取 TapTap 各类榜单，覆盖新游绿道和大盘老游。

**榜单 URL 映射：**

| rankType | 完整 URL | sourceTagCandidate |
|---|---|---|
| 预约榜 | `https://www.taptap.cn/top/reserve?page={page}` | 新游绿道 |
| 新品榜 | `https://www.taptap.cn/top/download/new?page={page}` | 新游绿道 |
| 热门榜 | `https://www.taptap.cn/top/download?page={page}` | 大盘老游 |
| 热玩榜 | `https://www.taptap.cn/top/played?page={page}` | 大盘老游 |

**品类子榜（可选）：** 在热门榜 URL 后追加品类 slug，如 `/top/download/action`（动作榜）、`/top/download/strategy`（策略榜）、`/top/download/acgn`（二次元榜）等 15+ 品类。

**输入：**

```json
{
  "rankType": "预约榜",
  "category": null,
  "maxPages": 3,
  "os": "android"
}
```

**页面解析规则：**

| 字段 | 提取方式 | 示例值 |
|---|---|---|
| rank | 列表序号 | `1` |
| gameName | `.taptap-top-card__name` 或链接文本 | `宗师之上` |
| appId | 链接 `/app/{appId}` 中提取 | `774700` |
| rating | 评分数字 | `8.2` |
| statusTag | 状态标签（测试/首发/公测） | `首发` |
| isLaunched | 是否已上线：预约榜固定 `false`，其他榜单从页面状态推断 | `false` |
| timeTag | 时间标签 | `2天前` |
| sourceUrl | `https://www.taptap.cn/app/{appId}` |

**输出：**

```json
{
  "source": "taptap",
  "rankType": "新品榜",
  "fetchedAt": "2026-05-22T14:00:00+08:00",
  "games": [
    {
      "rank": 1,
      "gameName": "宗师之上",
      "appId": 774700,
      "rating": 8.2,
      "statusTag": "首发",
      "isLaunched": false,
      "timeTag": "2天前",
      "sourceTagCandidate": "新游绿道",
      "sourceUrl": "https://www.taptap.cn/app/774700"
    }
  ]
}
```

#### `seo_game_rank_3839_timeline_fetcher`：预约时间线采集

**用途：** 按日期抓取即将上线/测试的新游预约日历。**这是原方案没有的新能力。**

**URL：** `https://www.3839.com/timeline.html`
**sourceTagCandidate：** 新游绿道（预约榜性质）
**分页：** 无限滚动加载，需模拟滚动或直接请求更多数据
**筛选 Tab：** 全部 / 即将上线 / 即将测试 / 即将更新 / 独家

**核心价值：**
- 精确到"今天/明天/下周X/06-15"的上线时间
- 区分"正式上线/限量测试/预下载/前瞻直播"等事件类型
- 可用于提前 1~2 周布局新游内容

**页面解析规则：**

| 字段 | 提取方式 | 示例值 |
|---|---|---|
| date | 日期标题文本 | `05月15日 上周五` |
| gameName | 游戏名 emphasis 文本 | `夜幕之下-全员恶人复仇题材(官服)` |
| detailUrl | 链接 `//www.3839.com/a/{id}.htm` | `//www.3839.com/a/182365.htm` |
| tags | 分类标签 | `["多平台", "角色扮演", "女性向"]` |
| rating | 评分数字 | `9.0` |
| eventText | 事件描述文本 | `定档6月5日正式上线,6月3日预下载` |
| actionType | 操作按钮文本（下载/预约/试玩/￥12） | `预约` |

**eventType 推断规则：**

| eventText 关键词 | eventType |
|---|---|
| 正式上线 / 已正式上线 | `launch` |
| 限量测试 / 开启测试 | `test` |
| 预下载 / 定档 | `preview` |
| 更新 / 版本 | `update` |

**输入：**

```json
{
  "tabFilter": "即将上线",
  "dateFrom": "2026-05-22",
  "dateTo": "2026-06-05",
  "maxItems": 100
}
```

**输出：**

```json
{
  "source": "3839",
  "rankType": "预约时间线",
  "fetchedAt": "2026-05-22T14:00:00+08:00",
  "timeline": [
    {
      "date": "2026-05-22",
      "events": [
        {
          "gameName": "七界梦谭",
          "detailUrl": "https://www.3839.com/a/164920.htm",
          "tags": ["多平台", "修仙", "开放世界"],
          "rating": 9.1,
          "eventText": "11:00 限量测试",
          "eventType": "test",
          "actionType": "预约",
          "sourceTagCandidate": "新游绿道",
          "sourceUrl": "https://www.3839.com/a/164920.htm"
        }
      ]
    }
  ]
}
```

#### `seo_game_rank_3839_hot_fetcher`：飙升/热门/预约/下载/二次元/玩家排行采集

**用途：** 抓取好游快爆 6 个榜单（飙升榜+热门/预约/下载/二次元/玩家），替代七麦安卓飙升榜及安卓应用商店畅销榜。

**榜单 URL 映射：**

| rankType | 完整 URL | sourceTagCandidate |
|---|
| 飙升榜 | `https://www.3839.com/top/sugar.html?page={page}` | 新游绿道 |
| 热门榜 | `https://www.3839.com/top/hot.html?page={page}` | 高转平替 |
| 预约榜 | `https://www.3839.com/top/expect.html?page={page}` | 新游绿道 |
| 下载榜 | `https://www.3839.com/top/down.html?page={page}` | 高转平替 |
| 二次元榜 | `https://www.3839.com/top/tag.html?page={page}` | 新游绿道 |
| 玩家榜 | `https://www.3839.com/top/player.html?page={page}` | 大盘老游 |

**页面解析规则：** 6 个榜单页面结构一致。飙升榜/热门榜/下载榜/玩家榜字段含 `downloadCount`（如 `3922下载`）；预约榜字段为 `reserveCount`（如 `10.4万预约`）。

**输入：**

```json
{
  "rankType": "热门榜",
  "maxPages": 5
}
```

**输出：** 同飙升榜结构，`sourceTagCandidate` 按 URL 映射表确定。

### 平台 C：B站游戏中心（`game.bilibili.com`)

**域名：** `https://game.bilibili.com`
**渲染方式：** ⚠️ **动态 SPA**（Vue），需 Puppeteer/Playwright headless 浏览器渲染
**翻页：** 滚动加载或"加载更多"按钮
**速度：** ~3s/页，建议作为补充源，非主力

#### `seo_game_rank_bilibili_fetcher`

**用途：** 抓取B站游戏中心新游榜，补充二次元/年轻向垂直品类。

**URL：** `https://game.bilibili.com/platform/ranks`
**sourceTagCandidate：** 新游绿道

**抓取方案对比：**

| 方案 | 说明 | 推荐度 |
|---|---|---|
| Puppeteer/Playwright | 启动 headless 浏览器，等待数据加载后提取 | ⭐⭐⭐ 推荐 |
| 拦截 XHR | 监听页面网络请求，找到数据 API 直接调用 | ⭐⭐ 需逆向 |
| SSR 降级 | B站游戏中心无 SSR，无法用 cheerio 直接解析 | ❌ |

**Puppeteer 抓取流程：**

1. 启动 headless Chrome，打开 `https://game.bilibili.com/platform/ranks`
2. 等待 `.rank-list` 或游戏卡片元素出现（`waitForSelector`）
3. 提取游戏卡片中的：游戏名、评分、标签、详情链接
4. 如需翻页，点击"加载更多"按钮或滚动触发

**输入：**

```json
{
  "rankType": "新游榜",
  "limit": 50
}
```

**输出：**

```json
{
  "source": "bilibili",
  "rankType": "新游榜",
  "fetchedAt": "2026-05-22T14:00:00+08:00",
  "games": [
    {
      "rank": 1,
      "gameName": "洛克王国：世界",
      "rating": 7.7,
      "tags": ["角色扮演", "宠物", "回合制"],
      "sourceTagCandidate": "新游绿道",
```

### 替代工具详细定义

#### `seo_game_rank_3839_hot_fetcher`：好游快爆 多榜采集

**用途：** 抓取好游快爆 6 个榜单（飙升/热门/预约/下载/二次元/玩家），替代七麦安卓飙升榜及安卓应用商店畅销榜。

**输入：**

```json
{
  "rankType": "飙升/热门/预约/下载/二次元/玩家",

  "limit": 100
}
```

**输出：** `sourceTagCandidate` 按 rankType 映射（飙升→新游绿道，热门/下载→高转平替，预约/二次元→新游绿道，玩家→大盘老游）。

**抓取方式：** 各榜单页面均为静态 HTML，cheerio 直接解析。

**输入：**
