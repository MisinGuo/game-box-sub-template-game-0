### 月度判断口径

- `14 ~ 21 天目标`：至少 `2` 个仓位出现稳定 Impression / Query
- `30 天目标`：至少 `1` 个仓位形成“主题页 → 问题页 → FAQ/对比页 → 主站承接页”闭环
- `首月继续条件`：标题改稿后有提升空间、站点边界仍清晰、主站承接页有点击
- `首月冻结条件`：7 仓位整体无 Impression、关键词持续越界、主站无承接点击
