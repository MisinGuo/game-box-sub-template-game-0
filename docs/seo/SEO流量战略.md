# 🔥 一、总战略（最终版）

> ✅ **SEO流量系统 + 转化系统 + 多站扩张系统**

------

## 👉 核心模型

```
流量获取：问答 + 破解截流 + 话题（关键词矩阵）
↓
流量承接：游戏页(/games/[id]) + 排行榜(/rank/)
↓
权重循环：0.1折专题(/01zhe/) + Tag聚合页(/tag/[tag]/)
↓
精准转化：游戏盒子(/boxes/) — 对比折扣、点击推广链接
↓
扩张复制：子站 Next-web-sub（品类垂直站）
```

> 💡 **本站核心差异**：不是通用游戏站，而是**游戏盒子折扣聚合平台**。
> 转化动作 = 用户点击盒子推广链接（CPS）或在盒子注册游戏（CPA）。
> 所有内容、排行、专题最终都要导向盒子下载/注册。

# 🧠 三、主站 vs 子站 战略分工（核心升级🔥）

------

## 🟢 主站（流量中心）—— `Next-web`（我爱玩游戏网 / 5awyx.com）

```
定位：游戏盒子折扣聚合站
核心词：游戏盒子、0.1折游戏、手游折扣、首充折扣
目标：吃「折扣 + 盒子 + 游戏名 + 问答」相关长尾词
作用：引流 → 游戏详情/盒子 → 转化（点击推广链接/注册）
```

> ✅ 主站核心是"盒子对比"，文章内容都要往"怎么找最划算的游戏/盒子"上靠

------

## 🔴 子站（垂直转化站）—— `Next-web-sub`（品类精细站）

```
定位：单一品类垂直内容站（卡牌/仙侠/传奇等）
核心词：[品类]游戏推荐、[品类]攻略、[游戏名]怎么玩
目标：吃细分品类词 + 玩法攻略词
作用：深度内容 + 精准用户 → 导流至主站盒子
```

> ✅ 子站代码已有现成框架（Next-web-sub），通过环境变量切换品类和内容配置

------

## 👉 一句话理解：

```
主站：捕鱼网（广，盒子折扣场景）
子站：鱼枪（深，品类攻略场景）
```

| 对比维度 | 主站（5awyx.com） | 子站（Next-web-sub） |
|---------|---------|---------|
| 核心词类型 | 盒子、折扣、0.1折 | 品类名+攻略 |
| 转化动作 | 点盒子推广链接 | 跳转主站盒子页 |
| 文章侧重 | 问答/真假/省钱 | 玩法/阵容/攻略 |
| 内容深度 | 广而浅 | 窄而深 |
