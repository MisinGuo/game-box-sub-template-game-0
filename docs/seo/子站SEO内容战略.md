# 子站 SEO 内容战略指南

> 适用站点：`Next-web-sub`（品类精细子站）  
> 核心目标：通过搜索引擎持续获取精准游戏用户，导流至主站完成转化

---

## 一、核心结论：游戏类型 + 攻略，不要只做攻略

### 为什么单纯发攻略不够？

| 维度 | 仅发攻略 | 游戏类型 + 攻略 |
|------|---------|----------------|
| 关键词覆盖 | 只覆盖"XX游戏攻略"类词 | 同时覆盖"仙侠游戏推荐"、"好玩的放置游戏"等大流量词 |
| 内容权威性 | 零散文章，缺乏结构 | 品类页作为"Hub"，建立垂直领域权威 |
| 内链结构 | 各文章孤岛，权重难以集中 | 类型页→攻略页 双向内链，权重流动清晰 |
| 用户意图覆盖 | 只覆盖「如何玩」 | 同时覆盖「玩什么」+「如何玩」 |
| 站点深度 | 浅，Googlebot 快速抓完离开 | 深，蜘蛛有大量有价值页面可抓 |
| 对抗竞争 | 容易被单篇文章替代 | 品类权威站不易被单页超越 |

**结论：子站的定位就是「品类垂直站」，必须同时建设「品类聚合页」和「攻略内容页」，二者缺一不可。**

|---------|-----------|------------|---------|

| 评测 `/reviews/[slug]` | "XX游戏值得玩吗"、"XX游戏评测" | 500-3k/篇 | E-E-A-T信号，提升站点权威 |

## 三、攻略内容策略

### 3.1 优先内容类型（按SEO回报排序）

**第一优先级：游戏新手攻略（转化最高，流量最稳）**
- 标题模板：`《[游戏名]新手入门攻略：开局必做X件事》`
- 核心词：`XX游戏新手攻略`、`XX怎么玩`、`XX新手教程`
- 长度：2000-4000字
- 必须覆盖：角色选择、开局资源分配、每日任务、避坑指南

**第二优先级：阵容/职业攻略（高需求，高复访）**
- 标题模板：`《[游戏名]最强阵容推荐2024：[具体场景]》`
- 核心词：`XX阵容推荐`、`XX用什么阵容`、`XX职业选择`
- 特点：版本迭代后需更新，用户收藏率高

**第三优先级：资源获取/零氪攻略（高流量，用户黏性强）**
- 标题模板：`《[游戏名]零氪攻略：免费获取[稀缺资源]的X种方法》`
- 核心词：`XX免费礼包`、`XX零氪玩法`、`XX礼包码`
- 特点：可以直接在文章中插入主站礼包领取链接，导流转化最直接

**第四优先级：版本更新/最新资讯（保持活跃度信号）**
- 标题模板：`《[游戏名][版本号]更新内容详解》`
- 作用：让蜘蛛频繁抓取，保持站点新鲜度，带动老文章重新排名

**第五优先级：横向对比/选择指南（大流量词，竞争激烈但值得做）**
- 标题模板：`《2024年最好玩的仙侠游戏TOP10推荐》`
- 核心词：`仙侠游戏推荐`、`好玩仙侠手游`
- 特点：流量大，但需要站点积累一定权重后才能排进前页

### 3.2 内容质量标准（Google E-E-A-T 要求）

```
E - Experience（亲历性）：文章需体现真实游戏体验，截图/数据来自实际游玩
E - Expertise（专业性）：覆盖深度信息（伤害公式、养成性价比、资源获取路径）
A - Authoritativeness（权威性）：建立作者信息页，显示作者游戏经验
T - Trustworthiness（可信度）：准确的数据，版本日期，有来源的引用
```

**每篇攻略文章必须包含：**
- [ ] 游戏名 + 具体主题（标题中体现）
- [ ] 版本/日期标注（让用户知道内容是否过时）
- [ ] 目录（H2/H3结构清晰，方便蜘蛛抓取大纲）
- [ ] 至少1张截图（图片alt标签写关键词）
- [ ] 末尾导流链接（指向主站该游戏详情页）
- [ ] 相关攻略推荐（内链到同游戏的其他攻略）

### 3.3 内容发布节奏

```
初期建站（0-3个月）：保量阶段
- 目标：每个子站积累 100+ 篇攻略
- 节奏：每天 3-5 篇（可以是短攻略500-1000字）
- 重点：覆盖该品类下每款主流游戏的新手攻略

成长期（3-6个月）：提质阶段
- 目标：重点游戏攻略 2000+ 字，覆盖更多长尾词
- 节奏：每天 1-2 篇高质量文章 + 1-2 篇资讯
- 重点：阵容攻略、零氪攻略（高商业价值词）

成熟期（6个月+）：扩词阶段
- 目标：品类聚合词（"仙侠游戏推荐"）开始排名
- 节奏：持续更新，重点更新历史旧文（版本变动时）
- 重点：横向对比文章、年度推荐榜单
```

## 四、URL 和页面结构优化

### 4.1 URL 规范

```
✅ 推荐：
/guides/mobile-legends-beginner-guide          # 英文slug，语义清晰
/guides/xx-youxi-xinshou-gonglue               # 拼音slug（中文站可选）
/games/immortal-journey                         # 简洁游戏slug

❌ 避免：
/guides/123456                                  # 纯数字ID，无语义
/guides/2024/04/01/xx-youxi-gonglue            # 日期在URL中，易过时
/guides/guide-article-post-xxx                 # 冗余词汇
```

### 4.2 面包屑结构（必须实现）

每个页面都需要：
```
首页 > 攻略 > XX游戏攻略 > 新手入门
```
- 面包屑提升点击率和排名理解
- 对应 `BreadcrumbList` JSON-LD（已在技术方案中实现）

### 4.3 内部链接策略（Hub-and-Spoke 模型）

```
                 ┌─────────────┐
                 │  首页 (Hub) │  ← 权重最高
                 └──────┬──────┘
                        │
         ┌──────────────┼──────────────┐
         ↓              ↓              ↓
   ┌─────────┐    ┌─────────┐    ┌─────────┐
   │ 游戏列表│    │ 攻略列表│    │ 评测列表│  ← 品类Hub页
   └────┬────┘    └────┬────┘    └────┬────┘
        │              │              │
   ┌────┴────┐    ┌────┴────┐    ┌────┴────┐
   │游戏详情 │←──→│攻略详情 │←──→│评测详情 │  ← 内容页（互相关联）
   └─────────┘    └─────────┘    └─────────┘
```

**关键规则：**
- 每篇攻略**末尾**必须有「相关攻略」3-5篇（同游戏或同类型）
- 每篇攻略**开头**必须链接到对应「游戏详情页」
- 游戏详情页**侧边栏**展示该游戏的所有攻略列表
- 首页**精选攻略**区展示近期高质量内容，保持权重流动

## 六、关键词选择策略

### 6.1 关键词分层模型

```
层级1：品类词（月搜1w+，难度极高，需要6-12个月积累）
  → "仙侠游戏推荐"、"好玩仙侠手游"、"仙侠游戏大全"
  → 主要靠首页和游戏列表页承接

层级2：游戏+类型词（月搜1k-1w，难度中等，3-6个月可排上）
  → "仙侠 MMO"、"横版仙侠游戏"、"仙侠放置"
  → 靠游戏列表筛选页 + 分类标签页承接

层级3：具体游戏词（月搜100-5k，难度较低，1-3个月）
  → "梦幻西游攻略"、"诛仙手游新手"
  → 靠游戏详情页 + 攻略详情页承接

层级4：长尾功能词（月搜<500，难度低，1个月内可排上）
  → "梦幻西游零氪玩法"、"诛仙手游阵容2024"、"XX游戏礼包码"
  → 靠具体攻略文章承接（最容易出效果，初期重点）
```

**建站初期：100%精力放在层级3和4，不要去抢层级1和2的词。**

### 6.2 标题优化技巧

```
❌ 差标题（关键词堆砌）：
"仙侠游戏攻略 梦幻西游攻略 新手教程 技巧 2024"

❌ 差标题（信息量不足）：
"梦幻西游攻略"

✅ 好标题（关键词+价值点+数字）：
"《梦幻西游新手入门攻略2024：7天快速升满级路线图》"

✅ 好标题（解决痛点型）：
"梦幻西游零氪攻略：每月免费领取价值500元资源的完整指南"

技巧：
- 标题前30字必须包含核心关键词
- 数字（7天、5件事、TOP10）提升点击率
- 年份（2024）表示内容新鲜度
```

## 七、与主站协同的 SEO 策略

### 7.1 子站→主站导流设计

子站每篇攻略必须包含1-2个主站链接（不能太多，避免被判断为广告站）：

```
位置1：文章开头（引导型）
"本文攻略基于最新版本，游戏注册和礼包领取请前往：
→ [游戏盒子主站 - XX游戏专区](https://www.gamebox.com/games/xxx)"

位置2：文章末尾（转化型）  
"📦 领取专属礼包 | 🎮 游戏官方下载 | ⭐ 查看更多推荐
→ [游戏盒子 - 仙侠游戏专区](https://www.gamebox.com/games/category/xianxia)"
```

### 7.2 主站反向支持子站

- 主站品类页添加「精选攻略入口」，链接回子站
- 链接加 `rel="nofollow"` 避免权重倒流（子站给主站传权重，不反向）
- 主站游戏详情页：「游戏攻略」板块链接到子站对应攻略（followed，支持子站权重）

### 7.3 品牌统一性（影响信任度）

- 子站和主站的 Logo/视觉风格保持一致
- 子站底部注明「本站由游戏盒子出品」（增加可信度）
- 保持相同的隐私政策和版权声明链接

## 八、内容运营规避风险

### 8.1 避免的SEO黑帽行为

```
❌ 批量生成低质量文章（AI不加工直接发布）
   → Google的 HCU（Helpful Content Update）会降整站权重

❌ 关键词堆砌（在正文中不自然地重复关键词20+次）
   → 可能触发手动处罚

❌ 隐藏文字/链接
   → 直接永久拉黑

❌ 购买垃圾外链
   → 比没有外链更差
```

### 8.2 推荐的白帽增长

```
✅ 在游戏官方社区/贴吧 分享真实攻略，自然带来外链
✅ 创建高质量的"游戏大全"型内容，会被其他站引用
✅ 与游戏玩家社区合作，互换友情链接（同类游戏站）
✅ 每次版本更新时更新旧文章（Google喜欢更新内容）
```

## 九、执行优先级总结

### 第一阶段（立即执行，技术基础）
1. 修复 6 个技术SEO问题（见第五节清单）
2. 确保每篇文章都有正确的 Article JSON-LD + BreadcrumbList
3. robots.txt 正确配置（禁止爬取API路由、管理页面）
4. 提交 sitemap 到 Google Search Console + Bing Webmaster

### 第二阶段（内容填充，建站1-3个月）
1. 每个子站至少 100 篇攻略（以新手入门和零氪攻略为主）
2. 为每款主流游戏创建独立的「游戏详情页」
3. 建立内链网络（每篇文章3-5个内链）
4. 添加作者页面，提升E-E-A-T信号

### 第三阶段（内容深化，3-6个月）
1. 将热门文章扩写至3000+字，补充截图和数据
2. 开始创作「横向对比」和「年度TOP10」型文章（冲击品类词）
3. 建立「专题合集」页面（类似Pillar Content），汇聚相关攻略
4. 监控 Google Search Console 数据，针对排名第8-20位的词重点优化

### 第四阶段（流量放大，6个月+）
1. 分析高流量攻略的用户行为，优化导流链接位置
2. 对排名稳定的文章进行内容更新（保持新鲜度）
3. 复制成功模式到其他品类子站
