# M11：追踪矩阵更新

> **这份材料回答：** 发布完后台记录在哪里？下次怎么防止重复选题？
> **输入：** M10 发布完成的文章信息（site_key、游戏名、角度类型、发布日期、文章 ID）
> **输出：** 内容追踪矩阵更新完毕，M6 去重索引同步
> **操作文件：** [内容追踪矩阵.md](../内容追踪矩阵.md)

---

## 发布后需要更新的三处

### 1. 素材卡「已完成的文章角度」

找到对应游戏的素材卡，把刚发布的角度 `⬜` 改成 `✅ | 发布日期`：

```
【已完成的文章角度（去重索引）】
✅ 截流型 | 2026-04-15          ← 刚发布，从 ⬜ 改成这个
⬜ 痛点解决型
⬜ 充值避坑型
...
```

### 2. 去重索引末尾追加

格式：`site_key|游戏名|角度类型`，每行一条

```
card-a|率土之滨|截流型
card-a|率土之滨|痛点解决型
```

> 这个索引直接复制给 M6 今日任务筛选的 `published_index` 变量用。

### 3. 发布统计表更新

找到对应游戏行，更新「已发篇数」「剩余篇数」「最近发布日期」：

```
| 游戏名   | 已发篇数 | 剩余篇数 | 最近发布日期 |
| 率土之滨 | 3       | 5       | 2026-04-15  |
```

5. 更新统计表的已发篇数和最近发布日期
6. 保存

## 方案二：程序自动更新

**适用：** 和 M10 API 发布联动，发布成功后自动记录

```python
import re
from datetime import date
from pathlib import Path

MATRIX_FILE = Path("docs/seo/内容追踪矩阵.md")

def update_tracking_matrix(site_key: str, game_name: str, angle_type: str):
    """发布成功后自动更新追踪矩阵"""
    today = date.today().strftime("%Y-%m-%d")
    content = MATRIX_FILE.read_text(encoding="utf-8")

    # 1. 把素材卡中对应角度的 ⬜ 改成 ✅
    old = f"⬜ {angle_type}"
    new = f"✅ {angle_type} | {today}"
    content = content.replace(old, new, 1)  # 只替换第一次出现

    # 2. 在去重索引区追加新条目
    index_marker = "<!-- 去重索引 -->"
    new_entry = f"\n{site_key}|{game_name}|{angle_type}"
    content = content.replace(index_marker, index_marker + new_entry)

    MATRIX_FILE.write_text(content, encoding="utf-8")
    print(f"追踪矩阵已更新：{site_key}|{game_name}|{angle_type} → ✅ {today}")
```

> `内容追踪矩阵.md` 的格式需要在去重索引区加一行 `<!-- 去重索引 -->` 注释作为程序插入锚点。

## 提取去重索引（供 M6 使用）

每次运行 M6 前，从追踪矩阵提取当前站点的所有 ✅ 条目：

**人工方式：** 在 `内容追踪矩阵.md` 里 Ctrl+F 搜索 `✅`，把每行的「游戏名 + 角度类型」组合整理成：

```
card-a|率土之滨|截流型
card-a|率土之滨|痛点解决型
card-a|原神|充值避坑型
```

**程序方式：**

```python
import re

def extract_published_index(matrix_path: str) -> list[str]:
    content = Path(matrix_path).read_text(encoding="utf-8")
    # 匹配格式：游戏名：XXX ... ✅ 角度类型
    # 根据实际矩阵格式调整正则
    games = re.findall(r"游戏名：(.+)", content)
    angles = re.findall(r"✅ (.+?) \|", content)
    # 简化版：直接从去重索引区提取
    index_section = content.split("<!-- 去重索引 -->")[-1]
    lines = [l.strip() for l in index_section.splitlines() if "|" in l]
    return lines
```
