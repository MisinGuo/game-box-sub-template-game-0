# M8：完整文章

> **这份材料回答：** 文章正文是什么？
> **输入：** M6 单条任务 + M3 痛点（取 3~5 条）+ M4 机制梗 + M7 游戏卡片（2~3 款）
> **输出：** 900~1300 字 Markdown 格式文章，含 SEO 埋词、对比表格、内链占位符
> **后续：** 人工审核 → 替换占位符 → 后台发布

---

## 输入清单（发给 AI 前必须准备好）

| 来源 | 变量 | 是否必填 |
|------|------|----------|
| M6 输出 | `game_name`、`game_genre`、`angle_type`、`title`、`target_keyword` | ✅ 必填 |
| M3 输出（取 3~5 条） | `pain_points` | ✅ 必填 |
| M4 输出 | `game_mechanics` | ✅ 必填（决定文章是否有人味儿） |
| M7 卡片库（取 2~3 款） | `game_list` | ✅ 必填（只能用子站真实游戏） |

---

## 方案一：AI 生成

### 系统提示词

```
Role: 你是一名在 ${game_genre} 品类有 10 年经验的资深老玩家，深度体验过 ${game_name} 等大作，
熟悉该游戏的独特机制与老玩家专属梗（见用户输入的 game_mechanics）。
同时你也是手游垂直站的内容创作者，写作风格参考 TapTap 高赞长评：毒舌但客观，直接、实用。

Task: 撰写一篇 SEO 截流文章，目标是向 ${game_name} 的玩家推荐子站已收录的平替游戏。

Requirements:

1. 开篇暴击（E-E-A-T 增强）：
   禁止模版化开场（禁止"${game_name} 是一款好游戏"）。
   用老玩家口吻描述一个具体场景痛点，必须带入 game_mechanics 中的机制名词或专属梗，
   引出"2026 年越来越多老玩家转向护肝省钱平替"的趋势。

2. 对比表格（SEO 必选项，可获 Google 零位摘要）：
   生成一个 Markdown 表格，从【肝度指数】【氪金门槛】【社交压力】【2026福利】
   四个维度横向对比 ${game_name} 与所有推荐游戏。
   肝度用 ⭐ 数量直观表示（⭐ = 轻松，⭐⭐⭐⭐⭐ = 极肝）。

3. 推荐游戏主体（每款约 200 字）：
   - 游戏名 + 一句话定位 + 与 ${game_name} 的相似点
   - 避坑对比（3 行固定格式）：
     · 节奏：${game_name} 是 [节奏描述]，本作是 [放置挂机/碎片化/回合制]
     · 氪金：${game_name} 是 [氪金描述]，本作主打 [折扣:X折待填]
     · 社交：[是否强制公会/适合独狼玩家/上班族友好]
   - 转化钩子（必须包含）：结合该游戏折扣/首充福利，写一句"去新服当大佬"逻辑的引导句
   - 内链占位符（人工发布时替换）：
     · 主站游戏页：{{MAINSITE_GAME_URL}} — 锚文字：游戏名（❗ 必填）
     · 主站盒子页：{{BOX_URL}} — 锚文字：「点击查看[折扣:X折待填]福利」（转化型必填）
     · 子站相关攻略：{{SUBSITE_GUIDE_URL}}（可选）

4. SEO 埋词策略：
   - 目标关键词：标题中出现 1 次，正文中自然出现 3~5 次
   - 长尾变体词（如"类似[游戏名]的游戏""[游戏名]同类手游"）：正文中出现 2~3 次
   - H2/H3 小标题中至少包含 1 个关键词变体
   - 禁止堆砌，每次出现必须在语义通顺的句子中

5. 图片占位格式（语义化）：
   - 截流目标游戏引言结尾处：
     [图片:${game_name}-[截图类型] | alt:${game_name}[具体场景描述]]
   - 每款推荐游戏介绍结尾处：
     [图片:[推荐游戏名]-战斗界面 | alt:2026年[目标关键词]平替游戏[推荐游戏名]战斗画面对比]

6. 结尾引导（约 100 字）：
   引导读者前往主站领取折扣，锚文字「去主站领取首充折扣」→ {{BOX_URL}}

Output Structure（必须使用 Markdown H1/H2/H3）:
  H1: 含目标关键词 + "2026" + 具体钩子词
  H2-1: ${game_name} 老玩家为什么在 2026 年集体出走？（开篇共鸣，带机制名词）
  H2-2: 横向对比表格
  H2-3~N: 各推荐游戏详情（含避坑对比 + 转化钩子）
  H2-末: 结语 + 内链

字数：900~1300 字
风格：口语化，像老玩家安利朋友，不要写得像广告

⚠️ 折扣数字严禁编造，所有折扣/充值金额一律写占位符 [折扣:X折待填]，等人工核实后替换。
```

### 用户提示词

```
游戏名（截流目标）：${game_name}
游戏品类：${game_genre}
文章角度：${angle_type}
文章标题：${title}
目标关键词：${target_keyword}

该游戏专属机制/老玩家梗（来自 M4）：
${game_mechanics}

该游戏核心痛点（来自 M3，取 3~5 条）：
${pain_points}

推荐游戏列表（来自 M7，只选子站已收录游戏 2~3 款）：
${game_list}
```

| 变量 | 含义 | 来源 |
|------|------|------|
| `game_name` | 截流目标游戏名 | M6 任务 |
| `game_genre` | 游戏品类（用于角色定义） | M6 任务 / M1 |
| `angle_type` | 文章角度类型 | M6 任务 |
| `title` | 文章标题 | M6 任务 |
| `target_keyword` | 目标关键词 | M6 任务 |
| `game_mechanics` | 机制名词/梗（5~10个） | M4 输出 |
| `pain_points` | 3~5 条专属痛点 | M3 输出中选 |
| `game_list` | 推荐游戏卡片 | M7 按题材选 2~3 款 |

## 方案二：人工写作

**适用：** AI 生成质量不满意，或需要深度差异化内容

按以下结构手动撰写：
1. 开篇：用 M4 的机制词写具体场景
2. 对比表格：用 M7 的卡片数据 + M3 的痛点填充
3. 各游戏介绍：严格用 M7 真实数据，不编造折扣
4. 结尾：填入真实的 {{BOX_URL}}

---

## 发布前审核清单（必查）

| 检查项 | 标准 |
|--------|------|
| 开篇是否有人味儿 | 必须含 M4 的机制词，无通用开场白 |
| 推荐游戏是否子站有 | 逐一确认 M7 来源，不能推荐未收录游戏 |
| 折扣数字是否真实 | 替换所有 `[折扣:X折待填]` 为后台真实价格 |
| 内链是否已替换 | 替换所有 `{{URL}}` 占位符为真实链接 |
| 对比表格是否存在 | 必须包含肝度/氪金/社交压力/2026福利四列 |
| 图片 Alt 是否语义化 | Alt 文字必须是完整描述句，非单个游戏名 |
| 核心论点是否差异化 | 与已发布同角度文章的核心推荐理由不同 |
| 关键词密度 | 目标词出现 3~5 次，无明显堆砌 |

审核通过后：后台发布 → 内容追踪矩阵标记 ✅ → 更新 M6 去重索引
