# M9：Meta Description

> **这份材料回答：** 搜索结果里的摘要文字写什么？
> **输入：** M8 文章标题 + 目标关键词
> **输出：** 110~150 字符的 Meta Description（中文约 50~70 字）
> **后续用途：** M10 后台发布时填入 Meta Description 字段

---

## 为什么不能直接用 AI 截取文章内容

Meta Description 直接决定搜索结果点击率（CTR）。AI 原文截取的描述往往：
- 缺少行动号召（CTA）
- 钩子词不够前置（前 50 字最重要）
- 字符数不受控（百度截断约 78 个汉字）

---

## 输出格式（标准化）

```
率土之滨玩累了？2026年老玩家都在转战这几款三国策略平替！主打0.1折开局，
肝度比率土低一半，更适合上班族。点击查看最新同类游戏平替清单。
```

规则：
- 总长度：110~150 字符（中文 50~70 字）
- 前 50 字必须包含「省钱/护肝/折扣」等核心钩子
- 必须包含目标关键词
- 末尾以行动号召（CTA）结尾

---

## 方案一：AI 生成

**适用：** 快速产出，再人工微调字数和钩子词

### 系统提示词

```
你是一名搜索结果 CTR 优化师，专门为手游推广站撰写 Meta Description。

请根据用户提供的文章标题和目标关键词，撰写 Meta Description，要求：
1. 总长度控制在中文 50~70 字（约 110~150 字符），超出会被搜索引擎截断
2. 前 15 字必须出现核心钩子词（省钱/护肝/低肝/折扣/平替/懒人 等其一）
3. 中间部分自然融入目标关键词（不能生硬堆砌）
4. 末尾以行动号召结尾，如「点击查看」「立即了解」「现在就去」
5. 语气口语化，像朋友推荐，不像广告文案

输出格式：
Meta Description：[内容]
字符数：[数字]
```

### 用户提示词

```
文章标题：${title}
目标关键词：${target_keyword}
文章核心钩子（省钱/护肝/折扣等，取一个最强的）：${hook}
```

| 变量 | 含义 | 来源 |
|------|------|------|
| `title` | M8 生成的文章标题 | M8 输出 |
| `target_keyword` | 目标关键词 | M6 任务 |
| `hook` | 这篇文章最强的转化钩子 | 如：0.1折、每天10分钟 |

## 方案二：模板填充（推荐，3分钟以内）

**适用：** 按角度类型套模板，改动最小

| 角度类型 | 模板 |
|----------|------|
| 截流型 | `${game}玩累了？2026年老玩家都在转战这几款${genre}平替！主打${hook}，策略性不减，更适合${audience}。点击查看平替清单。` |
| 痛点解决型 | `${game}太肝？上班族已经转向这几款低肝${genre}手游，每天${time}分钟搞定日常，还有${hook}福利。立即查看。` |
| 充值避坑型 | `${game}月卡值不值？老玩家转向这款${hook}充值更划算，攻略+折扣对比一篇说清楚。点击了解。` |
| 平替推荐型 | `找${game}同类平替？这${n}款${genre}手游策略性相近，肝度更低，${hook}开局当大佬，2026年最新推荐。` |
| 入坑决策型 | `${game}2026年还值得玩吗？老玩家现身说法，分析优缺点+推荐更省心的${genre}平替，${hook}。` |
| 零氪攻略型 | `${game}零氪能玩多久？数据测评告诉你上限在哪，还有${n}款同类游戏${hook}可以试试。` |
| 时间友好型 | `上班族能玩${game}吗？每天${time}分钟够不够用？这${n}款低肝${genre}手游${hook}，真正适合打工人。` |
| 跑路停服型 | `${game}2026年还稳吗？运营数据分析+推荐${n}款稳定运营${genre}平替，${hook}。` |

填写后用字符计数工具验证长度（推荐：[字数统计 - 在线工具]）。

## 方案三：程序生成

**适用：** 批量为多篇文章生成 Meta Description

```python
TEMPLATES = {
    "截流型": "{game}玩累了？2026年老玩家都在转战这几款{genre}平替！主打{hook}，策略性不减，更适合{audience}。点击查看平替清单。",
    "充值避坑型": "{game}月卡值不值？老玩家转向这款{hook}充值更划算，攻略+折扣对比一篇说清楚。点击了解。",
    # ... 其余角度
}

def generate_meta_description(angle_type, game, genre, hook, audience="上班族", time="10", n=3):
    tmpl = TEMPLATES.get(angle_type, TEMPLATES["截流型"])
    desc = tmpl.format(game=game, genre=genre, hook=hook, audience=audience, time=time, n=n)
    if len(desc) > 150:
        desc = desc[:147] + "..."
    return desc
```

## 质量检验

- [ ] 字符数 110~150（中文 50~70 字）
- [ ] 前 15 字有钩子词（护肝/省钱/折扣/平替 等）
- [ ] 包含目标关键词至少 1 次
- [ ] 末尾有行动号召（点击查看 / 立即了解）
- [ ] 没有出现 AI 编造的折扣数字
