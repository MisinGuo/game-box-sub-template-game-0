# Step 5：正文发布与回写

这个目录承接最后一段执行链：准备真实卡片、写正文、写 Meta、后台发布、回写追踪。

包含材料：

1. [M7-子站游戏卡片库.md](./M7-子站游戏卡片库.md)
2. [M8-完整文章.md](./M8-完整文章.md)
3. [M9-Meta描述.md](./M9-Meta描述.md)
4. [M10-后台发布.md](./M10-后台发布.md)
5. [M11-追踪矩阵更新.md](./M11-追踪矩阵更新.md)

员工进入本目录后的顺序：

1. 先从 M7 取真实卡片
2. 用 M8 生成正文
3. 用 M9 生成 Meta
4. 按 M10 发布到后台
5. 按 M11 回写 [内容追踪矩阵.md](../../03-执行/内容追踪矩阵.md)

这一步是人工复核最重的一步，折扣、版本、链接、SITE_ID 都不能跳过检查。
