# M5：文章角度矩阵

> **这份材料回答：** 针对这款游戏，可以写哪些文章，每篇文章的角度和标题是什么？
> **输入：** M1 `game_name` + M2 `keywords` + M3 `pain_points`
> **输出：** N 行文章任务卡（每行 = 一篇文章）
> **后续用途：** M6 今日任务筛选的 `task_matrix` 输入

---

## 输出格式（标准化）

```
| 角度类型 | 文章标题 | 核心论点 | 对应搜索词 |
|----------|----------|----------|------------|
| 截流型 | 2026年类似率土之滨的SLG手游推荐，这3款上班族最爱 | 低肝度平替推荐 | 类似率土之滨的游戏 |
| 痛点解决型 | 率土之滨太肝了？这几款国战SLG每天10分钟就够 | 时间压力解决方案 | 比率土之滨肝度低的游戏 |
| 充值避坑型 | 率土之滨月卡值不值？老玩家转向这款打折更多 | 首充性价比对比 | 率土之滨充值值不值 |
```

8 种角度类型：截流型 / 痛点解决型 / 充值避坑型 / 平替推荐型 / 入坑决策型 / 零氪攻略型 / 时间友好型 / 跑路停服型

### 系统提示词

你是一名 SEO 内容策划，专门为手游推广站生产高转化文章选题。

⚠️ 重要规则：每篇文章的标题核心词必须来自用户输入的"真实搜索关键词"中的某个词，不得自行发明搜索词。

请根据用户提供的游戏名、关键词表、痛点清单，生成该游戏所有可写的文章角度，按以下 8 类分别列出，每类给出 2~3 个标题：

每类标题要求：

输出格式（表格）：
| 角度类型 | 文章标题 | 核心论点 | 对应搜索词 |
|----------|----------|----------|------------|

## 方案三：程序生成

**适用：** 多款游戏批量生成，标题从模板填充关键词

## 角度类型速查

> 选题/内链决策时快速查阅。

| 角度类型 | 目标用户 | 转化点 | 内链重点 |
|----------|----------|--------|----------|
| 截流型 | 正在玩/搜大热门游戏的用户 | 同类游戏推荐 → 子站攻略 | 主站游戏页（必填）+ 主站盒子页 |
| 痛点解决型 | 已经不满当前游戏的用户 | 解决痛点的替代游戏 | 主站游戏页（必填）+ 主站盒子页 |
| 充值避坑型 | 准备充钱的用户 | 官服 → 盒子首充转化 | 主站盒子页（核心）+ 主站游戏页 |
| 平替推荐型 | 主动在找换游方向的用户 | 推荐列表 → 子站攻略 | 多个主站游戏页 + 子站相关攻略 |
| 入坑决策型 | 观望中、未决策的用户 | 打消顾虑 → 降低门槛 | 主站盒子页（零门槛试玩）+ 主站游戏页 |
| 零氪攻略型 | 不想花钱但想玩的用户 | 推动下载 + 盒子注册礼包 | 主站游戏页 + 主站盒子注册礼包链接 |
| 时间友好型 | 轻度玩家、上班族 | 推荐低时间成本游戏 | 主站游戏页 + 子站相关攻略 |
| 跑路/停服型 | 担心游戏稳定性的用户 | 推荐稳定运营游戏 | 主站游戏页（必填）+ 主站盒子页 |

```python
TITLE_TEMPLATES = {
    "截流型":    ["2026年类似{game}的{genre}手游推荐，{hook}", "喜欢{game}的玩家必看，这{n}款平替更省心"],
    "痛点解决型": ["{game}太肝了？这几款{genre}每天{time}分钟就够", "不想在{game}氪金？这款平替{hook}"],
    "充值避坑型": ["{game}月卡值不值？老玩家转向这款首充{hook}", "2026年{game}充值攻略：这样买才不亏"],
    "平替推荐型": ["有没有类似{game}但更省肝的游戏？推荐这{n}款", "{game}同类手游推荐，比原作{hook}"],
    "入坑决策型": ["{game}2026年还值得玩吗？老玩家现身说法", "{game}好玩吗？下载前先看这篇"],
    "零氪攻略型": ["{game}零氪能玩多久？这才是正确姿势", "不充钱玩{game}能走多远，数据说话"],
    "时间友好型": ["上班族能玩{game}吗？每天{time}分钟够不够", "{game}肝度测评：这类人不适合玩"],
    "跑路停服型": ["{game}2026年还稳吗？会停服吗", "{game}还值得玩吗？现在入坑晚不晚"],
}

def generate_task_matrix(game_name, keywords, genre, hook=""):
    rows = []
    for angle, templates in TITLE_TEMPLATES.items():
        for tmpl in templates:
            title = tmpl.format(game=game_name, genre=genre, hook=hook, n=3, time=10)
            # 从关键词表中匹配最接近的词
            matched_kw = match_keyword(title, keywords)
            rows.append({"角度类型": angle, "文章标题": title, "对应搜索词": matched_kw})
    return rows
```

> 程序生成的标题需要人工润色，确保标题自然通顺。
