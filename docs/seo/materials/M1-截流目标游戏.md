# M1：截流目标游戏

> **这份材料回答：** 这一轮内容生产要截流哪些游戏？
> **输入：** 子站的品类方向
> **输出：** 10 款处于衰退期、有截流价值的游戏列表
> **后续用途：** M2、M3、M4 的 `game_name` 输入

---

## 输出格式（标准化，供后续材料使用）

```
| 游戏名 | 品类（细分） | 衰退原因 | 截流角度建议 |
|---|---|---|---|
| 率土之滨 | SLG 策略 | 运营趋于保守，赛季感不强 | 主打「上班族不想铺路的平替」 |
| ... | ... | ... | ... |
```

---

## 方案一：AI 生成

**适用：** 快速开始，不需要查资料

### 系统提示词

```
你是一名手游市场分析师，熟悉国内安卓手游的生命周期规律和玩家社区动态。

请根据用户指定的游戏品类，列出目前处于「衰退期」的手游，筛选标准如下：
1. 曾是国内热门游戏（百度指数或 TapTap 评价数峰值较高）
2. 当前仍有稳定搜索量，但运营方已大幅缩减广告投入
3. 玩家社区内频繁出现「玩腻了/找平替/氪不动了」等讨论

列出 10 款符合条件的游戏，每款输出：
游戏名 | 品类（细分） | 衰退原因（一句话） | 截流角度建议（一句话）

要求：
- 只列用户指定的品类
- 优先列国产手游
- 截流角度建议要具体（例如：主打「上班族肝度低的平替」方向）

输出格式（表格）：
| 游戏名 | 品类（细分） | 衰退原因 | 截流角度建议 |
|---|---|---|---|
```

### 用户提示词

```
目标品类：${category}
```

| 变量 | 含义 | 示例 |
|------|------|------|
| `category` | 子站定位的游戏品类 | 卡牌/策略 |

## 方案二：人工选定

**适用：** 已经有想法、或想精确控制选题方向

1. 打开 TapTap → 分类 → 选目标品类 → 按「热度」排序
2. 筛选条件：评价数 10 万+ 但近期更新减少 / 差评率上升
3. 参考贴吧/B站UP主动态：哪些游戏玩家在大量发「弃坑帖」
4. 按照输出格式手动填表

---

## 方案三：程序生成

**适用：** 批量处理、定期自动更新候选列表

**思路：** TapTap 游戏列表按热度排序，抓取评分 + 近期评论量变化，计算衰退指数

```python
# 伪代码示意
# 1. 抓取 TapTap 指定品类 Top 100 游戏
# 2. 对比 30天前/90天前 的评论增量
# 3. 评论增量下降超过 40% 且总评价数 > 5万 → 列为候选
# 4. 输出标准格式表格

games = fetch_taptap_games(category="策略")
declining = [g for g in games if g.review_growth_30d < -0.4 and g.total_reviews > 50000]
```

> **注意：** TapTap 无公开 API，需用 requests + HTML 解析，或使用浏览器自动化。

## 人工确认清单

拿到候选列表后，对照子站已录入游戏列表，勾选：

- [ ] 子站有该游戏品类的推荐游戏可以承接（没有推荐游戏 = 截不住流量）
- [ ] 游戏名有真实搜索量（百度搜索框有下拉联想）
- [ ] 选 3~5 款进入本轮生产，不要一次选太多
